Instruções:
1-Conjuntos para treinamento e classificação estão contidos na pasta Conjunto.
2-carregar os dados para o treinamento da pasta treinamento (contida na pasta Conjunto).
3-para classificação, usar os arquivos da pasta classificação (contida na pasta Conjunto).
4-A variável que aloca os valores esperados para cada elemento do
  conjunto de treinamento deve ter o mesmo número de elementos que o 
  conjunto de treinamento, ou seja, para cada n itens do conjunto de 
  treinamento devem ter n respostas esperadas.

obs*
-Classe Principal do projeto: ExecutorMLP.
-Classe de classificação de portas lógicas: ClassificacaoPortasLogicas.
- Para o executor, a maioira das coisas já estão definidas,
  bastando somente passar o caminho para realizar um teste.
- os arquivos contidos em Conjunto são dados de características de   imagens já extraídos e tratados
